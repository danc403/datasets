Pokedex Associative Recall Dataset

Domain: Synthetic Pokedex Data (Gen 1-9)

Format: JSONL with Deterministic Masking (mask\_pre, mask\_target, mask\_post)

Training Purpose: Designed specifically for testing Context-Aware Inference and Associative Recall in Gated Linear Recurrent architectures.



Utility: Includes the conversion script pokemon.py to allow users to modify masking logic or adjust the "Lore-to-Stats" cross-referencing.
